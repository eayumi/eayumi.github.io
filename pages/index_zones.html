This section is an attempt on correlating the answers of the questionnairs with the actual observed data of the Mock-up.
The user can define intervals in which to divide the possible positions of the rotational elements. An example division is already given in the input fields below.
The bar charts below show how long the element was used in an interval. to be able to compare the single and pairs effectively, the height of a bar shows the relative duration of use (between 0 and 1) instead of the absolut duration (in ms). Especially the difference of use of the rotatable wall bewteen single and pairs of test-subjects might give further insights.
To update the charts click the corresponding Submit button. This will also draw the respective elements in the floor-plan with the given angles.
Keep in mind to delimit the angles with commas. The range of rotation for each element is given next to its name.
The chart can also be viewed by weekday or total week. This can be changed by the dropdown selection given below the input fields.
As a reminder, the English translation of the elements are: (Drehwand - Rotatable Wall), (Drehschrank - Rotatable Closet), (Lampe Drehwand - Lamp A), (Lampe Aussenwand - Lamp B)
