This plot shows for every movement that was measured for the corresponding element, from which 'From' (brightness_1) angle to which 'To' (play_arrow) angle it travelled.
While the green symbols show that the previous 'To' and next 'From' value match, the red values show that they mismatch.
The vertical lines show angle of some interesting positions. Furthermore,the horizontal gridlines are colored accoring to daytime:
00:00:00 - 06:00:00
06:00:00 - 12:00:00
12:00:00 - 18:00:00
18:00:00 - 24:00:00
At times, very small mismatchings are not severe. In order to set a tolerance, set a delta such that we have:
Two movements m_i and m_{i+1} are considered to be mismatched if abs([m_i].To - [m_{i+1}].From) smaller delta
